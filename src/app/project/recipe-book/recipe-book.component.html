Challenges Faced

One notable challenge encountered during the development process was implementing the forget password feature, specifically in creating a seamless multi-step process. The challenge was addressed by designing a step-by-step approach using an enum field. Each step, such as email validation, OTP submission, and new password setup, was managed by conditionally rendering views through the ngIF directive. This innovative solution allowed for a structured and user-friendly experience, overcoming the initial hurdles in the process.
