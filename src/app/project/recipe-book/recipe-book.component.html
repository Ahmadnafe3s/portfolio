Home Needs

The HomeNeeds is a web application design for Recipe Management and Shopping List. Leveraging technologies like Bootstrap and Angular for front-end design and Firebase for back-end functionality, this project divided into two section recipe book all and shopping list that offers a user-friendly and robust solution.

Challenges Faced

I had a couple of issues that I encountered recently. First, I needed to return the emitted data of an observable in the canActivate guard in order to dynamically prevent unauthorized or unauthenticated users from accessing certain routes. However, I found that in Angular, I couldn't return data inside the subscribe method. After some research, I discovered that I could use exhaust map to return the data instead of using subscribe, which solved the problem.

Another issue I faced was displaying the logo in the login or logout form section. Despite trying various solutions, I couldn't get the logo.png file to be recognized inside the auth component, regardless of where I kept it. Eventually, I found a solution on Stack Overflow - I needed to keep the logo.png file in the assets folder. Once I did that, the image was successfully displayed.
